# 📚 Book Kingdom

A simple web app to manage and browse books. Built with Tailwind CSS and modern web tools.

## Features
- Add, edit, and delete books
- Interactive UI with hover effects
- Cart system
- Responsive design
- Search
- Add to Cart

## 🛠️ Tech Stack
- HTML
- Tailwind CSS
- JavaScript
- React
- mysql
- axios

## 📦 Installation
```bash
git clone [https://github.com/MulkyAyyubi/upk-mulky]
cd backend
npm start
cd frontend
npm run dev
